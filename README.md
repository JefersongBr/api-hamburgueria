# API Hamburgueria

## Sobre o projeto

API desenvolvida em C# usando ASP.NET Core para gerenciar uma hamburgueria.

## Funcionalidades

* Cadastro de clientes
* Cadastro de produtos
* Criação de pedidos

## Tecnologias

* C#
* ASP.NET Core
* Entity Framework (em desenvolvimento)



## Aprendizado

Projeto desenvolvido para prática de desenvolvimento de APIs 
com foco em Programação Orientada a Objetos, organização de código e boas práticas.
